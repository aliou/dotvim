# Vim settings.

## Install
    
    git clone git://github.com/aldl/dotvim.git ~/.dotvim
    ln -s ~/.dotvim/vim ~/.vim
    ln -s ~/.dotvim/vimrc ~/.vimrc
    cd ~/.dotvim
    git submodule update --init

## Epitech

If you are an Epitech student, remember to change these [lines][l1] with your login
and name.
